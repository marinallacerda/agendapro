AgendaPro

Sistema de agendamento profissional para salões, clínicas e serviços em geral

🚀 Como iniciar

1

Dê duplo clique em iniciar.bat

Abre automaticamente o servidor (porta 3001) e a interface (porta 5173) em duas janelas separadas.

2

Acesse no navegador

O sistema vai abrir automaticamente em http://localhost:5173

3

Crie sua conta

Clique em "Criar agora" e preencha seu nome, nome do negócio, email e senha.

⚠️ Mantenha as duas janelas abertas (servidor e interface) enquanto usar o sistema.

📋 Fluxo de uso

1

Configure seus serviços

Menu Serviços → adicione manicure, pedicure, lazer, limpeza de pele, etc. com nome, duração e preço.

2

Configure seus horários

Menu Horários → ative os dias da semana e defina o horário de atendimento. Bloqueie datas específicas.

3

Configure PIX e WhatsApp

Menu Configurações → aba PIX: adicione sua chave. Aba WhatsApp: configure a Evolution API.

4

Compartilhe seu link de agendamento

No Dashboard, copie seu link e envie para clientes. Eles vão ver os serviços, escolher horário e pagar pelo PIX.

📱 WhatsApp — Configuração Opcional

Para enviar lembretes automáticos, você precisa da Evolution API (gratuita e open source).

1

Instale a Evolution API

Siga a documentação em github.com/EvolutionAPI/evolution-api

2

Configure no sistema

Menu Configurações → aba WhatsApp → preencha URL, API Key e instância → ative os lembretes desejados.

💡 Os lembretes são enviados automaticamente: 24h antes e 1h antes do agendamento.

💳 PIX — Como funciona

Após o cliente agendar, o sistema gera automaticamente um QR Code e o código copia-e-cola do PIX.

O valor cobrado é o preço do serviço configurado. O QR Code é gerado seguindo o padrão oficial BACEN/EMVCo.

🌐 Usando em rede local (outros dispositivos)

Para acessar de celulares ou outros computadores na mesma rede Wi-Fi:

ipconfig → anote o IP local (ex: 192.168.1.100)

No celular, acesse: http://192.168.1.100:5173/agendar/seu-slug
